# Keyword Research Frameworks

## Volume Estimation (Without Tools)

When Ahrefs/Semrush isn't available, estimate relative volume:

| Signal | Interpretation |
|--------|---------------|
| Google Autocomplete suggests it | Medium-high volume |
| Multiple PAA questions about it | Medium volume, high intent |
| Reddit threads with 50+ comments | High interest, rising topic |
| Competitor has a dedicated page | Worth targeting |
| No competitor content exists | Either low volume or untapped opportunity |
| Google Trends shows upward curve | Growing — get in early |

## Topic Cluster Architecture

### Hub-and-Spoke Model
```
Pillar Page (2500-4000 words)
├── Cluster Article 1 (1500-2000 words) ← links to pillar
├── Cluster Article 2 (1500-2000 words) ← links to pillar
├── Cluster Article 3 (1500-2000 words) ← links to pillar
└── Pillar links to ALL cluster articles
```

### Content Silo Model
```
Category Page
├── Subcategory 1
│   ├── Article A
│   └── Article B
├── Subcategory 2
│   ├── Article C
│   └── Article D
└── Cross-links ONLY between related silos
```

## South African Keyword Patterns

- "in South Africa" / "in SA" / "in Joburg" suffixes common
- Afrikaans variants: "beste" (best), "goedkoop" (cheap), "naby my" (near me)
- Price queries use "R" or "rand" not "$"
- Load shedding creates seasonal spikes for: generators, UPS, inverters, solar
- Data/airtime queries spike around payday (25th-1st)
- "Near me" queries growing faster in SA than global average

## Competitive Gap Analysis Framework

1. **Content gap:** Topics competitors cover that you don't
2. **Quality gap:** Topics you both cover but their content is better
3. **Format gap:** Topics where a different format would win (video > text, tool > article)
4. **Freshness gap:** Topics where competitor content is outdated
5. **Entity gap:** Topics where competitors lack structured data/schema
